# 🧠 AI Resume Screening System

An AI-powered resume screening tool built with Streamlit that automatically ranks candidates against a job description using NLP and semantic similarity — with an AI-based validation layer for reliability.

🔗 **Live App:** https://resumescreener-hmd2aqrr3b2ubzhgdoecyv.streamlit.app/

## 📋 Overview

Recruiters often spend hours manually reviewing resumes for a single role. This system automates the first pass of screening by:
- Extracting text from resumes (PDF/DOCX)
- Identifying relevant skills
- Computing a semantic similarity score between the resume and job description
- Ranking multiple candidates side by side
- Validating results with an independent AI-based second opinion

## ✨ Features

- **Login system** — simple session-based authentication
- **Job Description input** — paste any JD as plain text
- **Multi-resume upload** — supports PDF and DOCX formats
- **Resume parsing** — extracts full text from uploaded resumes
- **Skill extraction** — identifies technical and soft skills using keyword matching
- **Semantic matching** — uses `sentence-transformers` to score contextual similarity between JD and resume
- **Candidate ranking** — combines semantic and skill match scores into a final ranking, sorted best to worst
- **Candidate detail view** — full breakdown per candidate: matched skills, missing skills, extracted text
- **AI validation** — an independent score and reasoning generated by Google's Gemini API, shown alongside the computed score to verify reliability
- **Analytics dashboard** — visual charts comparing candidates (overall score, semantic vs. skill match breakdown)
- **CSV export** — download the ranking table for offline use
- **Settings panel** — adjust scoring weight balance (semantic vs. skill match) and toggle AI validation on/off

## 🛠️ Tech Stack

- **Frontend/App:** Streamlit

- **Data handling:** pandas, NumPy

## 📊 How Scoring Works

Each candidate receives a **Final Score** calculated as:
- **Semantic Match** — measured using sentence-transformer embeddings and cosine similarity between the JD and resume text
- **Skill Match** — percentage of JD-mentioned skills found in the resume
- Weights are adjustable via the Settings sidebar (default: 60% semantic, 40% skill)

## ✅ Validation Approach

To ensure the scoring system produces reliable results, each candidate's match is independently re-evaluated by Google's Gemini model, which reads the same JD and resume and provides its own score and reasoning. This acts as a cross-check against the computed score, helping confirm that rankings reflect genuine candidate fit.
